Broadcast Add (Manage Broadcast)


This screen can be used to define a new broadcast message.

Name: Define the unique name for the Broadcast message.
Type: Select whether you wish to create an adhoc broadcast message from scratch or create a message based on an existing message template that was defined using the Manage Template function. If you wish to create based on a predefined template, choose Predefined Template else choose Define Now.

Following fields appear on choosing Predefined Template Option:

Template: This is a mandatory field. Choose the relevant template that you wish to use from the drop down list. The list contains all templates defined using Manage Template section(Ex : Seasonal Greetings Template).

Search By: Once the Template is chosen, it is time to choose the recipients of the message. Use the Search By option to decide the basis on which you wish to choose the recipients. (Ex : Customer category, Product, Service, Subscription, Subscriber Category)

Search Value : In Search value, provide the value for the chosen Search By option. Ex : If you had chosen the Search By as Product, then choose the product name from the Search Value list(Ex: HP Laptop). This will mean that you wish to send the current broadcast message to all the people who have purchased the product(HP Laptop) from your firm. This way you are able to send tailored messages or promotions only to the specific customer category.

When: Choose the Date and time by when you wish to send the message to the people in the chosen group. The Date and time should be some valid date and time in the future. Once the valid date and time is defined, the application will automatically schedule the message on the mentioned date and will send the message to all the people mentioned in those chosen criteria(i.e based on Search Value).

Once all these values have been provided, choose Finish or Cancel.


In case you have chosen Define Now option:

You will see the Search By, Search Value, When, message template type,subject and message fields.

Please find below the explanation for message template type, subject and message fields. Please refer the above sections for explanation for the rest.

Message Template type : Choose whether the current adhoc message is sent as SMS or Email.

Subject : Enter the text to appear in the Subject of the Email.

Message: Enter the Message that should be broadcasted to the chosen customer category. You can include ${receiver.fullname} with the message text to specify the name of the person receiving this message.

Finally after defining the broadcast message, you can choose the following:

Finish : Choosing this button will permanently store the defined broadcast message.

Cancel : Choosing this button will ignore the defined broadcast message and the control is taken to the Broadcast screen.
