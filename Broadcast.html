Broadcast (Manage Broadcast)


This screen can be used when ever you wish to broadcast or send bulk messages to a group of people or firms. These messages can be send either through an email or SMS. You can either send adhoc messages or send messages through a predefined Message template (Please Refer the Manage Template Section to understand more about creating Broadcast or Message Templates).

This screen typically lists all the messages that have been prepared for broadcasting and shows the current status (Pending, Sent etc). You can either edit, copy, delete or cancel any of these messages. Additionally you can also Add a new message to be broadcasted.

Inorder to add a new message, click on Add button at the right bottom corner of the screen.(Refer to the Broadcast-Add Section help to get more details on how to add a new message to be broadcasted).

You can also export or transfer the list of broadcast messages appearing in this screen to a Microsoft Excel(spreadsheet), CSV(comma separated text file), XML(Extensible markup language file) or Text (flat or plain text file). Choose the file format by pressing the respective button.

If you wish to modify or delete an existing message, you can perform the same through the following :

Point the cursor or mouse pointer towards any of the message item listed in the screen. Immediately, you will see the following actions namely :

Edit : Click this button if you wish to modify  the current message. Please note that only those messages that have not yet been broadcasted and those that does not have the status as Cancelled can be modified. (Editing or modification is same as Adding a Broadcast message, except that the current values are populated allowing to change to new values. Please refer the help section Broadcast-Add for more details on editing a broadcast message).

Copy: Click this button if you wish to copy the message settings to create another new broadcast message.(Please refer the help section Broadcast-Edit for more details on editing a broadcast message).

Delete : Click this button if you wish to remove the current message. You can only delete those messages that do not have the status as "Cancelled"

Cancel: Click this button if you wish to cancel the current message. In this case, a new popup screen will appear and you can enter the reason for cancellation. On pressing Ok, the message will be cancelled and you can see the status changing to "Cancelled". You cannot edit an

Instead of choosing any of these buttons, if you click on any of the Broadcast message, then the Broadcast View Screen opens(Read Only mode) and displays the contents of the chosen broadcast message. You can also cancel the Broadcast Message using the Cancel Button in the View Screen.
