Adding External Appointments into the Calendar
 Previous Top Next

You can add external appointments to the calendar in order to manage and track all your events at one place.

You must specify the following details:

·    Specify a Subject and a Description for your external appointment.

·    Select a Date from the calendar.

·    Specify a Start Time and End Time. You can either enter the time or select the time using the navigation arrows. 

·     Click Add to save the details.
