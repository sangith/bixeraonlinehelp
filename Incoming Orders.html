Incoming Orders(Customer - Incoming Orders)


This Screen lists all the Incoming Orders from Customers with the following details for each order

Customer Name,Created Date,Delivery Requested,Status(Order Status like Created, Accepted, Delivered etc), Payment Status(Pending, Paid etc) , Office(Facility that received the Customer Order),Product Summary(Products or Services Summary in the Order)

You can perform the following functions on the Customer Orders listed in this screen. Once you move the mouse pointer on any listed order, the relevant functions that can be performed on that Order gets displayed below the same. Please note that the Status of an Order determines the functions that can be performed on that Order :

Edit: This function can be used to modify the contents of the Customer Order. You can only modify those orders that have not yet reached the "Delivered" Status. Please Refer to the
Incoming Orders - Edit Section for more details on this function.

Order cancel: This function can be used to cancel a Customer Order. You can only cancel those orders that have not yet reached the "Delivered" Status. On clicking the Order Cancel button, you are prompted with a message to capture the purpose of cancellation. Provide the message and then click on Cancel Order if you wish to go ahead with cancellation or click Close to exit without cancelling the Order. Once you have cancelled an Order, the status of the Order is updated to "Cancelled". There are no other operations that you could perform on a Cancelled Order.


Accept: This function can be used to Accept a Customer Order. Accepting the Order is the first step for processing the Order. On clicking the Accept button, you are prompted with a message to capture the notes on the acceptance of the Order. Provide the details and then click on Accept Order if you wish to go ahead with acceptance or click Close to exit without accepting the Order. Once you have accepted an Order, the status of the Order is updated to "Accepted".


Reject: This function can be used to Reject a Customer Order. You can only reject an order that are with the status "Created" . On clicking the Reject button, you are prompted with a message to capture the notes on the rejection of the Order. Provide the details and then click on Reject  if you wish to go ahead with Order rejection or click Close to exit without rejecting the Order. Once you have rejected an Order, the status of the Order is updated to "Rejected". Once an Order gets the "Rejection" status there are no further operations allowed on this order.

Shipped: This function can be used to Update the shipping details of a Customer Order. You can only update the shipping details for those orders that are with the status "Accepted" . On clicking the Shipped button, you are prompted with details like Date Shipped, Shipment Id(Transporter/Lorry receipt number etc) and message to capture the notes on the shipment of the Order. Provide the details and then click on Shipped  if you wish to go ahead with updation of "Shipped" Status or click Close to exit without updating the status of the Order. If you have clicked the "Shipped" button then the status of the Order is updated to "Shipped".

Delivered: This function can be used to Update the Delivered details of a Customer Order. You can only update the Delivered details for those orders that are with the status Shipped . On clicking the Delivered button, you are prompted with details like Delivered Date, message to capture the notes on the Delivery of the Order. Provide the details and then click on Delivered  if you wish to go ahead with updation of "Delivered" Status or click Close to exit without updating the status of the Order. If you have clicked the "Delivered" button then the status of the Order is updated to "Delivered".

Send Quotation: Choose this function if you wish to upload a quotation for this Customer Order. Please Refer the Help Section on the Incoming Order - Quotation for more details

Create Invoice: Choose this function if you wish to create an invoice for this Customer Order.Please Refer the Help Section on the Incoming Order - Invoice for more details

Payment Request:This function can be used to make a Payment Request to the Customer for the current Customer Order. You can only update the Delivered details for those orders that are with the status Shipped or Invoiced . On clicking the Payment Request button, you are prompted with message to be sent to the Customer for the Payment Request. Mention the message details and then click on Request  if you wish to go ahead or click Close to exit without any action. There will be no change to the status of the Order. You will be able to send many such Payment Request


Paid : Choose this function if the Customer has made the payment for the current Order. On clicking the Paid button, you are prompted with message to be mentioned for the Paid Status. Mention the message details and then click on Paid  if you wish to go ahead or click Close to exit without any action. There will be no change to the status of the Order but the Payment Status gets updated as Paid.
