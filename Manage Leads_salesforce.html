Add Opportunitys (Salesforce - Manage Leads)


This screen is used to store the potential Opportunitys that the firm receives from different interested parties.

You can store Information, Contact Details, Notes about each Opportunity.

Opportunity Information :

Salutation: This is a drop down list. Choose the respective salutation (Ex: Mr., Mrs., Miss etc)

Name: This is a mandatory field. Enter the name of the person who has provided the opportunity.

Owner: The User id of the Owner of the firm is defaulted automatically.

Organization Name : This is a drop down list. If the Opportunity arises from a registered Prospective Organizaton (Added to the application through the Salesforce-Manage Organization functionality), then choose the Organization from the list.

Department: This is a drop down list. Choose the department that is providing the current opportunity, from the list.

Lead Source : This is a drop down list. Choose the Source through which the current lead or opportunity was received (Ex: Social media, Advertisement, Meeting, Employee Referral etc). This can help in future analysis of the most popular sources from which you receive the leads or opportunities.

Interested Products & Services: This is a drop down list. Choose from the List or enter the products and services that the prospect had shown interest. Enter 1 product or service at a time followed by a comma to add next product or service. Once you have entered all the products and services, press the tab key to move to the next field.

Tags: This is a drop down list. Choose the relevant Tag from the List.

Potential Closing Date: Choose the Date by when this opportunity will elapse if unaddressed(date/month/year Example : 22/10/2016).

Referred By Email : Enter the Email id of the person who has referred this opportunity or lead.

Referred By Profile : This is a drop down list. Choose the Profile/User Id of the person who has referred this opportunity or lead.


Existing Customer: This is a drop down list. If the Opportunity has been referred by an existing customer, then choose the customer from the list.

XYZ Profile: Choose the User id or User profile of the person from the Existing customer organization.

Description: Use this field to enter more details about the Opportunity (Ex: when is the proposal needed, what are the special terms and conditions etc).

There are 3 buttons namely - Previous(Disabled as this is the first screen of the function), Next and Cancel

Select Next button if you have completed the details in the current screen and wish to move to the next section (i.e the Contact Section).

Select Cancel if you wish to ignore the details entered in the current screen and do not wish to proceed to enter other details.


Contact Number : You can add the Contact numbers of the Opportunity by clicking the +Add button next to the contact number field.

On clicking the +Add button, 2 fields appear namely

Type: Choose the Contact Type from the drop down list (Example : Mobile/Home/Office) by clicking on the required value.

Number: Type the Contact Number for the Chosen Contact Type (Example : Mobile number in case the Contact Type chosen was Mobile). It is a Mandatory field. You can enter the complete contact number along with the Country,city code etc.

In this way, you can enter the various contact types along with the respective Contact numbers.

In case you wish to remove one of the contact details, point the cursor on the respective contact details. Immediately the application will display a Delete button below it. Click on the same to remove that contact details. This is a standard feature which works in a similar way for the next 2 fields mentioned below.

Contact Emails : Click the +Add Button next to the Contact Emails field to add contact email details of the Opportunity.

Type: Choose the Contact Emails Type from the drop down list (Example : Home/Office/Other) by clicking on the required value.

Email: Type the Email ID for the Chosen Contact Emails Type (Example : xyz@yahoo.com, the Home/Personal Email id in case the Contact Emails Type chosen was Home). It is a Mandatory field.

In this way, you can enter the various contact Email types along with the respective Email id.


Address: Click the +Add Button next to the Address field to add contact Address of the Opportunity. On clicking the +Add button, a new window popups up with the following details :

Type :  Choose the Contact Address Type from the drop down list (Example : Home/Office/Other) by clicking on the required value.
Field 1: This is a Mandatory field. Enter the Contact Address Details like the Door number, Building number, Street name etc.

Field 2: If there are more details that needs to be added, use the field 2 to continue with the Contact Address.

Field 3: If still more details are to be added beyond field 1 and field 2, you can use this field to add more Address details.

City: Enter the name of the City(Ex: Bangalore, Mumbai etc.). This is a mandatory field.

State: Enter the name of the state(Ex: Karnataka, Maharashtra etc).This is a mandatory field.

Country: Enter the name of the Country(Ex: India,Canada etc). You can choose from the Drop down list. This is a mandatory field.

Pin code: Enter the Pin code of the City(Ex : 500412 etc).

Click on OK button if you wish to store the Contact Address Details or Cancel if you wish to ignore the entered details. On clicking on either of the button, the Contact Address Add window closes and it takes you back to the Opportunity-Add Window.

Click the Next Button to provide additional details like the Departments, Relationship and Category.


There are 3 buttons namely - Previous, Next and Cancel

Select Previous button if you wish to review any information entered in the Previous screen (i.e the Opportunity-Information  Section).


Select Next button if you have completed the details in the current screen and wish to move to the next section (i.e the Opportunity- Notes Section).

Select Cancel if you wish to ignore the details entered in the current screen and do not wish to proceed to enter other details. On choosing Cancel, all details entered in the current screen and the previous (Opportunity- Information) section will be ignored and you will be taken to the List of Opportunities Screen.


Notes Section : You can use this section to enter the Notes pertaining to the opportunity.

Click the +Add button to add new notes.

On clicking the +Add button, a new (ADD) screen appears where you can add the Notes.

Notes : This is a mandatory field. Enter any note title/simple note descriptioin in this field.

Documents : Use this field to upload any relevant documents from your system. Inorder to add/upload documents, press the +Upload button present next to the Documents Section.

On clicking the +Upload button, a new field with the words Drop files to upload or click appears. Clicking on it opens the standard Windows file open screen. Use this screen to choose the relevant file by choosing the drive and the folder where the file is situated in your system. Click the Open button. This will upload and link the selected file to the current opportunity.

Press X to ignore and close this screen. This will delink the file uploaded from the opportunity and also ignore the note entered for the opportunity.

OR

Press "//" to confirm the linking of the file along with the Notes for the Opportunity.

There are 3 buttons namely - Previous, Finish and Cancel

Select Previous button if you wish to review any information entered in the Previous screen (i.e the Opportunity-Contact  Section).


Select Finish button if you have completed the details in the current screen and wish to store this information permanently in the system.

Added by : System stores the Current User id automatically for this opportunity.

Date : System stores the current date and time in the server for this opportunity.


Select Cancel if you wish to ignore the details entered in the current screen and do not wish to proceed to enter other details. On choosing Cancel, all details entered in the current screen and the previous (Opportunity- Information,Opportunity-Contact) section will be ignored and you will be taken to the List of Opportunities Screen.
