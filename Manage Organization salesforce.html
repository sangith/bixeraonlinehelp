Manage Organization(Sales Force)


This Function is used to add new prospective organization that may be interested in doing business with the firm.

Following Details about the Prospective Organization are captured :

Name : Name of the Prospective Organization. This value is mandatory.

Owner : Name of the User who is registering the Organization. You can either choose the profile of the Owner from the values in the drop down list or key in any name manually.

Website : Provide the website of the Prospective Organization.

Contact Number : You can add the Contact numbers by clicking the +Add button next to the contact number field.

On clicking the +Add button, 2 fields appear namely

Type: Choose the Contact Type from the drop down list (Example : Mobile/Home/Office) by clicking on the required value.

Number: Type the Contact Number for the Chosen Contact Type (Example : Mobile number in case the Contact Type chosen was Mobile). It is a Mandatory field. You can enter the complete contact number along with the Country,city code etc.

In this way, you can enter the various contact types along with the respective Contact numbers.

In case you wish to remove one of the contact details, point the cursor on the respective contact details. Immediately the application will display a Delete button below it. Click on the same to remove that contact details. This is a standard feature which works in a similar way for the next 2 fields mentioned below.

Contact Emails : Click the +Add Button next to the Contact Emails field to add contact email details of the Prospective Organization.

Type: Choose the Contact Emails Type from the drop down list (Example : Home/Office/Other) by clicking on the required value.

Email: Type the Email ID for the Chosen Contact Emails Type (Example : xyz@yahoo.com(Home/Personal Email id) in case the Contact Emails Type chosen was Home). It is a Mandatory field. You can enter the complete contact number along with the Country,city code etc.

In this way, you can enter the various contact Email types along with the respective Email numbers.


Address: Click the +Add Button next to the Address field to add contact Address of the Prospective Organization. On clicking the +Add button, a new window popups up with the following details :

Type :  Choose the Contact Address Type from the drop down list (Example : Home/Office/Other) by clicking on the required value.
Field 1: This is a Mandatory field. Enter the Contact Address Details like the Door number, Building number, Street name etc.

Field 2: If there are more details that needs to be added, use the field 2 to continue with the Contact Address.

Field 3: If still more details are to be added beyond field 1 and field 2, you can use this field to add more Address details.

City: Enter the name of the City(Ex: Bangalore, Mumbai etc.). This is a mandatory field.

State: Enter the name of the state(Ex: Karnataka, Maharashtra etc).This is a mandatory field.

Country: Enter the name of the Country(Ex: India,Canada etc). You can choose from the Drop down list. This is a mandatory field.

Pin code: Enter the Pin code of the City(Ex : 500412 etc).

Click on OK button if you wish to store the Contact Address Details or Cancel if you wish to ignore the entered details. On clicking on either of the button, the Contact Address Add window closes and it takes you back to the Organization-Add Window.

Click the Next Button to provide additional details like the Departments, Relationship and Category.

Departments : This is a drop down list. You can choose from the list or can manually add the departments in the Prospective Organization that conduct business with your firm. You can add more than 1 department in this field.

Relation Type: This is a drop down list. You can choose the type of Relationship of this prospective Organization with your firm from the list(Example : Investor, Customer,Partner or Others)

Industry Type: This is a drop down list. You can choose the type of Industry of this prospective Organization from the list(Example : Small Enterprise, Medium Enterprise, Large Enterprise or Others).

Category: This is a drop down list. You can choose the Category of this prospective Organization(Example : Private, Government, Subsidiary or Others).

Status : This is a drop down list. Define the Status of the Prospective Organization. Choose Active if you have an Active relationship with it or Choose Non Active if you don't have an active relationship.


Previous Button: Select the Previous Button if you wish to go back to the Previous screen to add/modify the Prospective Organization details.

Finish Button: Select the Finish button once you have entered all the details of the Prospective Organization and wish to permanently store these details in the application.

Cancel Button : Select the Cancel button if you don't wish to store any of the details.
