Message Template Add - (Manage Template - BroadCast)


You can use Message Template to create standardized and routinely used broadcast messages. These predefined templates can then be used by other users to easily and quickly broadcast messages resulting in standardization of the broadcast messages.

Name : Enter the name of the Broadcast message template (Ex : Seasonal Greeting message template, Thank you for visiting/ordering message template etc)

Type: Select whether the current message template is meant for broadcasting through SMS(short message service) or Email(Electronic Mail). Choose the required option from the Drop down list.

Subject:  This field appears only when the Broadcast Type chosen is "Email". You can mention the summary of your message in the Subject (Ex: Seasons Greetings)

Message: Enter the Standard Message that needs to be broadcasted. (Ex : Seasons Greetings to all our valuable customers). Typically the messages do not have reference to any name/person. However if you wish to include the name of the person, then you can add "${receiver.fullname} in your message.

At the time of broadcasting the message, the application will  insert the respective name of the person/firm from the chosen list (Customers/Investors etc) for each of the broadcast message.

Example: "Dear ${receiver.fullname}, We wish to convey our Seasons Greetings for the upcoming festival season. We Wish you and family abundance of happiness and good luck"  -In this case, the application will substitute with the name of the person for the corresponding email id/mobile number and if there are 3 names(Suresh, Ramesh, Rajan) in the list chosen during broadcast, the broadcast message will be like :

Message 1 : Dear Suresh, We wish to convey our Seasons Greetings for the upcoming festival season. We Wish you and family abundance of happiness and good luck"

Message 2: Dear Ramesh, We wish to convey our Seasons Greetings for the upcoming festival season. We Wish you and family abundance of happiness and good luck"

Message 3 : Dear Rajan, We wish to convey our Seasons Greetings for the upcoming festival season. We Wish you and family abundance of happiness and good luck"

Choose OK button to store the Broadcast Message Template in the system. On choosing OK, the system displays a message "Message Template Added Succesfully"

Choose Cancel if you do not want to store the Broadcast Message template.
