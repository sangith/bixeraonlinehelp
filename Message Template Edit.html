Message Template Edit


You can use this screen to edit or modify the Message Template contents already created.

You can change the Name, Type or message or all the 3. All the 3 fields are Mandatory fields, hence you cannot leave any item blank.

The values to be entered or modified are similar to those mentioned in the Message Template Add Screen. Please refer the Message Template Add
help item if you need assistance.

After modifying the contents of the Message Template contents, you can either decide to confirm the modifications by clicking the OK Button or ignore the changes by clicking the Cancel Button.
