Message Template


This Screen lists all the Message Templates that are created by the user and stored in the application.

There are 3 buttons namely Add and Export at the bottom right side of the screen.

Select Add when you wish to add new Message Templates.

Select Export when you wish to copy all the message templates into a Microsoft Excel(spreadsheet application) or CSV(Comma separated file), XML(Extensible Markup language file format) or Text(Plain text file).

When ever you wish to exit the screen, you can choose the "X"  button in the right bottom corner.


Additionally you can also remove all or chosen Message templates from the displayed list. Inorder to remove the message templates, you can select the check box appearing at the beginning of the respective Message templates you wish to remove. Other method of selecting the Message templates for removal is the "Select All" Option. This can be achieved by choosing the checkbox that appears in the section header list in front of "Name" section header. This action will choose all the Message templates in the list. You can see that all the checkbox in front of each of the Message template is selected.

Once the Message Template is selected for removal, you can see a "Dustbin" button at the bottom right side of the screen. You can click on the same to remove all the selected Message Templates. Once a message template is removed from the application, you cannot get it back. If needed, you need to recreate them.

Alternatively if you wish to remove the selected Message Templates one by one, then you can use the Delete Button that appears right below each of the listed Message Template. This button appears when you move the mouse pointer on any of the Message Template list.

In the similar way, if you wish to edit or modify any of the Message Template, you can use the Edit button that appear just below each of the Message Template list. The Edit button appears when you move the mouse pointer near any of the Message Template.

On choosing the edit button, the application takes you to the Message Template Edit Screen.
