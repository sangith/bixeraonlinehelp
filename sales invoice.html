Use the Quick Invoice screen to create simple customer invoices very easily.



Inorder to create a quick invoice, assign invoice number, choose the items in the invoice (with its quantity, discount etc), key in the total discount, applicable tax etc. Once all the details are provided, either Save as Draft (Temporary save) or proceed to Save it and print it or send to the Customer.


Invoice Number : The Invoice number is an auto-generated value which increments by 1 for each new invoice.

Customer : This field can be used to choose the customer for whom the invoice is being raised. All customers registered with the System using the "Add Customer" Option is listed here to enable the user to choose the right Customer.

Click the +Add button to start adding new items to the Invoice.

Items : The Items that are being invoiced are chosen from this section. Each item has fields like Category, Item ID, Item Description, Unit Price, Quantity, Base Price etc.


Select Category : In each invoice, you can include items that belong to any Item Category. The Item category field helps in easily finding the items that fall under the chosen category. In order to choose the Item Category, click the Drop Down List below the label "Select Category". Typing the first few characters of the Item Category description and pressing the "Search" button" can selectively display only those categories that already exist in the system and those that begins with the terms or characters used in the current search. You can navigate from top to bottom of the list by moving your mouse up and down. Select the required Item Category to choose the same for the invoice.

If you cannot find the required category in the list, you can also add a new Category from the same place. You can do this by choosing "New Item". On Choosing the "New Item" from the list, a new dialog box pops up named "Edit Item". This dialog box can be used to create a new item along with its key attributes like Item Name, Quantity, Price etc.

Once all the details are keyed in, press the "OK" button to use this item details in the invoice or "Cancel" button to ignore the newly created item.

Item ID : Key in the Item Identification Code of any Item that was already stored in the system. If any Item ID that does not exist in the system, then an error message gets displayed warning that the "Given Item ID is not associated with any Product". If any of the Existing Item Identification is chosen, then the details like "Item", "Unit Price" etc. are automatically displayed in the screen.

Quantity:  Choose the Sales Quantity for the selected Item.

Base Price : The Base Price gets automatically calculated by the System for the chosen Item. This is calculated by multiplying the Quantity sold with the Unit Price.

Discount : If you wish to provide any discount to this item, you can mention it in this field. You can mention the Discount value (Example : Rs.100). Alternatively if the discount is defined for this item in the Item Master, then that discount is automatically displayed for this item. You can proceed with the same amount or change the same.

Tax : In case any tax is applicable for the chosen item, it is displayed here from the Item Master.

Total : The Total Value for the current item is displayed here (Total = Base Price - Discount + Tax)

In the similar manner, you can add more items in the invoice. Click on the "Add" button to add additional items to the invoice.

In case you do not need any item that you have created in the Quick Invoice, you can use the "x" button (red color) next to the Total field of the respective item. This action removes the entire line item from the invoice.

To proceed with entry of other items, use the +Add button.

Once you have listed all the items in the invoice, if the Payment is collected at the same time, then you can check the "Payment Received" Checkbox. This action will automatically generate a receipt for the invoice. This facility can be used only when the customer pays the full amount of the invoice, while the invoice is generated.

Once the invoice is completed, you can press any one of the below mentioned buttons :

Send : Press this button when you have finalized the invoice and wish to send it to the Customer. Once this button is pressed, the Invoice details are saved in the application and it is marked to be shown in the "Dashboard Section (Status->Unpaid Invoice subsection)" when the Customer logs in.

Save as Draft : Press this button, if you have not finished with the invoice and you wish to make a temporary pause. Once you save the invoice as Draft, the invoice details gets stored in the application with "Draft" Status. When ever you wish to continue with the invoice, you can always retrieve this invoice and either modify/add new details to the invoice. On finalizing the invoice, you can press the "Send" button.
Save and Print: Press this button when you have completed the invoice and wish to save the details in the application and also print the invoice.  In this case, the Invoice will not be marked to be displayed in the "Dashboard Section" when the customer logs in.

Print: Press this button when you wish to take a printout of the invoice. The details will not get stored in the application. You can continue adding/modifying details in the invoice.


Close: Press this button when you wish to close the Quick invoice screen. If you have not pressed the "Save as Draft" or "Send" or "Save and Print", the details in the screen are ignored(i.e not saved) and the Quick Invoice screen gets closed and you will be taken back to the Sales Invoice Listing Section.
