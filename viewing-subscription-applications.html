Viewing Subscription Applications
 Previous Top Next

You can view the list of all subscription applications using this screen.

You can view the following:

1.      Picture: Displays the photograph of the customer.

2.      Name: Displays the name of the customer.

3.      Status: Displays the status. When you add an unregistered customer, the status is set to Active, by default. The status is Pending for Approval, when you link an unregistered customer with an existing customer and waiting for the customer to approve the process.

4.      Email: Displays the email address of the customer.

5.      Number: Displays the phone number of the customer.

You can also do the following:

·      Click at the bottom of the screen to perform various operations.

·      Click to add a registered or an unregistered customer. To add a registered customer to the coaching institute, click Add a registered Customer. You can only add the customers who have registered through the website. To register an unregistered customer, click Add an Unregistered Customer.
